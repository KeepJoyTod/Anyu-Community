# Anyu-Community
安隅智能社区是一个基于 Java 17 + Spring Boot + RuoYi 深度定制的现代化物业管理与居民服务平台。包含 Vue 后台管理系统与 UniApp 移动端，提供物业缴费、在线报修、访客邀请与社区活动等全方位数字化社区解决方案。
Anyu Smart Community is a modern, full-stack property management and resident service platform built with Java 17, Spring Boot, and UniApp. It provides a comprehensive digital solution for modern communities, featuring property bill payments, online maintenance requests, visitor management, venue booking, and community activities, effectively streamlining property operations and enhancing the living experience for residents.
