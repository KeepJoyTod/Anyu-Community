# 安隅智能社区 (Anyu Smart Community)

## 📖 项目简介
安隅智能社区是一个基于前后端分离架构打造的现代化、智能化社区管理与服务平台。项目致力于通过数字化手段提升物业管理效率，并为社区居民提供便捷的在线服务，如物业缴费、报修、社区活动、访客邀请等。

## 🏗️ 项目结构
本项目包含两个主要部分：

- **Behind/ (后端服务与后台管理系统)**
  - **后端**：基于 Java 17 和 Spring Boot 4.0.3 开发，采用 MyBatis-Plus、Druid、Quartz、JWT 等核心技术栈。架构基于 RuoYi 框架进行深度定制开发。
  - **后台 Web 端**：位于 `Behind/ruoyi-ui`，基于 Vue 2 和 Element UI，提供全面的后台业务管理界面。
- **Front/ (移动端用户前台)**
  - 基于 UniApp 和 Vue 开发的移动端应用（支持 H5 及微信小程序等），面向社区居民提供日常服务与交互入口。

## ✨ 核心功能模块

### 📱 居民服务 (Front 端)
- **物业服务**：物业费/水电费查询与缴纳、在线报修申请。
- **便民服务**：快递驿站查询、社区场地预约、访客通行证申请。
- **社区互动**：社区公告与通知、爱心活动、周边商城。
- **个人中心**：个人信息管理、历史账单及投诉建议。

### 💻 后台管理 (Behind 端)
- **业务管理**：
  - **费用管理**：物业费、水电费等账单的生成与核销。
  - **报修与工单**：居民报修单据的跟进与状态流转。
  - **通知与公告**：社区重大通知的编辑与发布。
  - **访客与场地**：访客记录审核、公共场馆预约审批。
- **系统管理**：用户管理、角色权限、菜单配置、部门及字典管理。
- **系统监控**：在线用户、操作日志、服务状态监控。

## 🛠️ 环境与技术栈要求
- **JDK**：17 (必须)
- **数据库**：MySQL (≥ 5.7)
- **缓存**：Redis
- **前端环境**：Node.js (≥ 14.x)
- **开发工具推荐**：IDEA (后端)、HBuilderX / Trae IDE (前端全栈开发)

## 🚀 快速启动

### 1. 数据库与缓存准备
1. 创建 MySQL 数据库，并执行项目根目录下的 `ry.sql` 脚本完成表结构与基础数据的初始化。
2. 启动本地 Redis 服务。

### 2. 启动后端 (Behind)
1. 使用 IDEA 打开 `Behind` 目录。
2. 检查 `ruoyi-admin/src/main/resources/application.yml` 等文件中的数据库和 Redis 连接配置。
3. 运行相关 Application 启动类，启动 Spring Boot 后端服务。

### 3. 启动后台管理前端 (Behind/ruoyi-ui)
1. 进入目录：`cd Behind/ruoyi-ui`
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`

### 4. 启动移动端前台 (Front)
1. 推荐使用 HBuilderX 导入 `Front` 目录。
2. 检查并配置 `config.js` 或相关网络请求工具中的后端 API 接口基础地址。
3. 运行到内置浏览器或微信小程序模拟器。

## 💡 编码规范与开发指南
根据项目约定，后端开发需遵循以下规范：
- **无 DTO 模式**：后端接口建议直接通过 `Map<String, Object>` 灵活返回结果，坚持使用纯 POJO，避免引入繁琐的 DTO 对象转换。
- **全栈联调**：推荐使用 Trae 进行前后端代码的快速编写与联调，将 Request 逻辑平滑注入到 UI 纯渲染组件中。
- **环境异常排查**：遇到任何版本相关错误时，优先确认当前的 JDK 17 及其他依赖版本环境是否匹配。
